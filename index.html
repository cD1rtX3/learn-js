Basic JavaScript Concepts

Introduction:

It is a commonly held belief that learning to code is a difficult task. However, quite the opposite is the case. Languages like JavaScript are built to be simple to understand and use, making learning easy with the right teacher.

Take this example code block:

let arr = [];

for (let i = 0; i < 10; i++) {
    arr[i] = i;
}

While it may look intimidating now, breaking it down into its simple constructs yields an easy to follow pattern, and by the end of this module, you should be able to tell what this code is doing.
